## Run Locally

Clone the project

```bash
  git clone https://github.com/DavidMPRocha/Netflix-Clone.git
```

Go to the project directory

```bash
  cd Netflix-Clone
```

Install dependencies

```bash
  npm install
```
Or
```bash
  yarn
```

Start project

```bash
  npm start
```
Or
```bash
  yarn start
```
